# COCOMO Media - Immersive 3D Website

A premium, immersive marketing website for COCOMO Media built with React, Three.js, and React Three Fiber. Features cosmic particle fields, floating 3D geometry, bloom effects, and smooth scroll animations.

## Features

- **3D Particle Field** - 4000+ particles with purple/violet/pink/cyan gradient
- **Floating Geometry** - Wireframe toruses, dodecahedrons, icosahedrons
- **Bloom Post-Processing** - Glowing, premium visual effect
- **Scroll-Triggered 3D Section** - Torus knot reveal when scrolling
- **Framer Motion** - Smooth animations and micro-interactions
- **Responsive** - Optimized for desktop and mobile

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- **React 18** + **Vite**
- **Three.js** - 3D graphics
- **React Three Fiber** - React renderer for Three.js
- **@react-three/drei** - Useful R3F helpers
- **@react-three/postprocessing** - Bloom & effects
- **Framer Motion** - Animations

## Deploy

Deploy to Vercel, Netlify, or any static host. The build output is in `dist/`.
